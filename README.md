# LPClash
Low poly unity 3d Rpg demo code

Rpg 3D game made on Unity.

This demo has got the following:
     - Inventory system
     - Items quality
     - Weapons
     - Shields
     - Equipment sets
     - Pickups
     - Combos
     - Quests
